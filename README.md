# calculadora-websocket

Aplicação client/server completa usando WebSocket na porta 3000

Descriação
----------
Este projeto é uma aplicação cliente/servidor que permite realizar operações matemáticas básicas (soma, subtração, multiplicação, divisão e potenciação) em tempo real utilizando WebSocket.

Tecnologias utilizadas
----------------------

*   HTML
*   CSS
*   JavaScript
*   Node.js
*   API Express
*   API WS

Como executar
-------------

1.  Clone este repositório em sua máquina:

git clone https://github.com/seu-usuario/calculadora-websocket.git

2.  Instale as dependências do projeto:

`npm install`

3.  Inicie o servidor:

`node server.js`

4.  Acesse a aplicação em seu navegador:

http://localhost:3000


Como utilizar
-------------

1.  Insira os valores e selecione a operação desejada.
2.  Clique no botão "Calcular".
3.  O resultado será exibido na tela.
